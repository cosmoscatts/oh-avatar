# Vue3 Tsx 练手项目

### 为什么会有这个项目？
最近在练习 `vue-tsx`，所以找了网上的开源项目练手，你可以在此处 [在线预览](https://oh-avatar.netlify.app/)。

这个项目~~参考~~抄写了 [Codennnn/vue-color-avatar](https://github.com/Codennnn/vue-color-avatar)，只是使用 `tsx` 重构了一遍。

### 项目启动
```bash
pnpm i
pnpm run dev
pnpm run build
```
